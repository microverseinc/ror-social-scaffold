# Stay in touch

In this stay in touch project we are about to create a simple website where users can share posts and create relationship between then. the project is still in the development process. 

## Built With

- Ruby on Rails
- Active Record
- Bootstrap 4
- Ruby

## Getting Started

To get a local copy of the repository please run the following commands on your terminal:

```
$ cd <folder>
```

```
$ git clone git@github.com:verissimohenry/Stay-in-Touch.git
```

```
$ cd ror-social-scaffold
```
## Prerequisites

Ruby: 2.6.3
Rails: 5.2.3
Postgres: >=9.5

## Setup

Instal gems with:

```
bundle install
```

Setup database with:

```
   rails db:create
   rails db:migrate
```

## Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.
